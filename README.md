# OpenTelemetry Astronomy Shop Demo

> A clean, easy-to-read fork of the OpenTelemetry demo project.

## Overview

This repository contains the OpenTelemetry Astronomy Shop Demo, a sample microservice-based system for learning distributed tracing, metrics, and observability with OpenTelemetry.

Use this demo to:

- explore OpenTelemetry instrumentation patterns
- run a realistic sample application locally
- test tracing and monitoring integrations
- learn how services communicate in a distributed system

## What’s included

- multiple sample microservices
- OpenTelemetry tracing and metrics support
- Docker deployment guides
- Kubernetes deployment examples
- observability-focused application traffic

## Quick start

1. Clone the repository:

```powershell
git clone https://github.com/iam-veeramalla/ultimate-devops-project-demo.git
cd ultimate-devops-project-demo
```

2. Review the deployment files and service definitions.
3. Follow the official docs for Docker or Kubernetes deployment.

### Run with Docker

Use the Docker deployment guide:

- https://opentelemetry.io/docs/demo/docker_deployment/

### Run on Kubernetes

Use the Kubernetes deployment guide:

- https://opentelemetry.io/docs/demo/kubernetes_deployment/

## Usage

After cloning, explore the repository structure, configuration, and sample applications. This demo is a great starting point for:

- learning OpenTelemetry collector configuration
- integrating tracing backends such as Jaeger or Zipkin
- experimenting with vendor observability platforms
- visualizing distributed service behavior
